JBIG-2 Bitstreams

The following JBIG-2 compliant bitstreams have been generated to exploit and test the various options and features in the JBIG-2 standard and our encoder.

All of these bitstreams have been independently verified. However, errors may still exist.

If you have a JBIG-2 decoder, then please try them out and let us know if you were successful.  Conversely, if you have any JBIG-2 compliant bitstreams you would like to share or have tested, then please send them to us as e-mail attachments (MIME preferred).

If errors are detected in any of these bitstreams, they will be corrected and updated.  The date of each bitstream is included to help identify new and/or updated bitstreams.

Also, if the description of the bitstream test case is unclear (or doesn't test what it is supposed to) then please let us know.

In the rightmost column, the Power JBIG-2 Encoder options are listed
